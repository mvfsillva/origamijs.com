What is
Origami.js?

Initially it's a tool for teaching geometry, web and javascript in schools. Currently it's a powerful library to create using HTML5 Canvas.

Why to use it?

Learning the canvas API can be challenging. But the learning curve might be reduced with a few simplifications: a chainable canvas, stylization of objects using familiar CSS notation, brings acessibility and resposible canvas, and easy access to the context using selectors.
